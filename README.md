# Student Result Management System

A Java-based Student Result Management System developed as part of our second-semester Object-Oriented Programming (OOP) course. This project demonstrates the application of core OOP principles, file handling, and Java Swing to manage academic records in a structured and user-friendly way.

---

## Overview

The Student Result Management System is designed to simplify the management of student academic information by providing an organized interface for handling student records, courses, transcripts, and fee details.

The project emphasizes clean object-oriented design, modular development, and persistent data storage using file handling techniques.

---

## Features

- Student record management
- Course management
- Transcript generation and viewing
- Fee record management
- Persistent data storage using file handling
- Java Swing graphical user interface
- Modular object-oriented architecture

---

## Tech Stack

- **Language:** Java
- **GUI Framework:** Java Swing
- **Concepts:** Object-Oriented Programming (OOP)
- **Storage:** File Handling
